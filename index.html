Contacts Application

C# | ASP.NET MVC | SQLite | HTML | CSS | JavaScript

The app takes advantage of dependency injection by using an IoC container to handle the newing up of objects Used a SQLite database on the backend for data storage. I created a ContactRepository model with CRUD methods for inserting new contacts, displaying contacts, updating a contacts, and deleting contacts. The ContactController links the models to the views and calls the correct views to display depending on the user’s interaction.

The BestBuy database simulates an inventory management system and employee sales records. I implemented a repository pattern to isolate the data access logic and business logic in order to allow for a more loosely coupled approach to data access from a sample BestBuy database. This project demonstrates CRUD operations through a console user interface
